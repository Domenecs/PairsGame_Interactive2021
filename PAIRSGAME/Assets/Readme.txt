Este es un proyecto que se realizó para la feria de Cocentaina de 2022. El proyecto inicialmente era un proyecto de aprendizaje en el que se hacía pareja de cartas (2017). Este se adaptó a posteriori para usarse con realidad Aumentada.

El proyecto utiliza los packages de Vuforia y de DotWeen, aunque al actualizar la versión de Unity el de Vuforia quedó obsoleto. Usese como quiera.
